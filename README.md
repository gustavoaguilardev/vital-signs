# Vital Signs Monitoring API

Spring Boot REST API for registering and querying vital signs data.

## Technologies
- Java 21
- Spring Boot
- Spring Data JPA
- MySQL
- OpenAPI 3 (Swagger)
- Maven

## Features
- Register vital signs data
- Query historical vital signs records
- RESTful API architecture
- API documentation with Swagger UI

## API Documentation
Once the application is running, access Swagger UI at:

http://localhost:8080/swagger-ui/index.html

## Project Origin
This project was developed as an academic project at ITSOEH and received institutional recognition.
